# PIDiff
PIDiff: Physics-inspired diffusion model with expert-aware prototype memory bank for fusion classification of hyperspectral and LiDAR data

## Code Availability

The complete code will be made publicly available after the paper is accepted.

## Dataset Links

The datasets used in this work can be downloaded from the following public sources:

- MUUFL Gulfport dataset: https://github.com/GatorSense/MUUFLGulfport
- Houston2013 dataset: https://machinelearning.ee.uh.edu/2013-ieee-grss-data-fusion-contest
